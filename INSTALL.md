## Requirements

### `Linux`

- `gcc`
- `make`

available in **`$PATH`**

### `macOS`

- `g++`
- `make`

available in **`$PATH`**

## Installation

```console
git clone git@github.com:daze-lang/daze.git /opt/daze
cd /opt/daze
sudo make
```

- The **`daze`** binary will be placed in **`/opt/daze/bin`**, and will be copied to **`/usr/bin`**
